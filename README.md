# Cokigalleta's Combat System For RP

Cokigalleta’s Combat System for RP es un sistema de combate web ligero diseñado para roleplay en plataformas como WhatsApp o Discord. Permite a los jugadores ejecutar acciones de combate desde una interfaz clara y centralizada, generando registros verificables que el administrador puede validar para evitar trampas y spam. El sistema está pensado para distintos canales o universos de rol, es totalmente funcional sin servidores y está optimizado para uso en dispositivos móviles.
